TesseractRAG

A fully deployed production RAG system with multi-session document Q&A. FastAPI backend on Railway, frontend on Cloudflare Workers, and persistent storage via Backblaze B2. Engineered to run under a 512 MB RAM constraint.

  • Integrated Llama-3.1-8B-Instruct via HuggingFace Inference API with rule-based query routing
  • Memory-efficient: lazy-loaded reranker, serialized FAISS indices, on-demand BM25 rebuilding
