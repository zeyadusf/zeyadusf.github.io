TesseractRAG

A production RAG system rebuilt from scratch around a dispatcher-based layered architecture — swap any component (embedder, retriever, LLM) from a single .env variable without touching the pipeline logic.

  • Language-aware hybrid retrieval: BM25 + pgvector HNSW + RRF fusion — forces semantic-only search on cross-language (Arabic/English) queries where lexical matching breaks down
  • Multi-provider generation via Llama-3.3-70B (Groq) and Llama-3.1-8B (HuggingFace) with dual query rewriting for semantic and lexical retrieval paths
  • Automated RAG evaluation using Cohere as LLM-as-a-Judge — scores faithfulness, answer relevancy, context precision, and context recall per session
  • Role-based admin dashboard with full user lifecycle controls and platform-wide analytics
